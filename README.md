# 🎬 Movie Jokes Project

## 📌 Description

This project is a Python program developed as part of a **Coursera Python course assignment**.

The program:

- Retrieves movie information from the OMDb API
- Extracts the Rotten Tomatoes rating
- Searches for jokes related to the movie plot
- Highlights matching words in both the plot and jokes
- Displays formatted output in the terminal

---

## 🛠️ Technologies Used

- Python 3
- requests
- re
- json (built-in library)

## 🔑 API Requirements

This project uses the **OMDb API**.

To run the project:

1. Get a free API key from:
   https://www.omdbapi.com/apikey.aspx

2. Insert your API key inside the script:

```python
OMDB_API_KEY = "YOUR_API_KEY_HERE"
```

## ▶️ How to Run

Make sure Python is installed.

1. Install requests (if not already installed):

2. pip install requests

3. Run the program:
  python move_jokes.py
